Remote Workers and Work from Home Employees

Description

With a large percentage of the workforce moving out of the more secure confines of the corporate network, the magnitude of Cyber Security risks increase. Compound this with the presence of malicious cybercriminals who bait the tiniest sliver of opportunity and you have a significant challenge in hand.

The Covid 19 pandemic has accelerated the shift to remote work and the trend suggests that this will continue. Considering the same, Security Quotient is pleased to offer this comprehensive course that is designed to equip the workforce to protect information and information assets, irrespective of where they work from.


Course Overview - Work safely from anywhere

This light-weight course gives an overview of Cyber Security risks while working outside the secure confines of the corporate network. Further, the course explains essential Cyber Security skills that every employee must gain. These skills help prevent malicious Cyber Attacks and keep valuable information and information assets safe.


What will you learn?

Cyber Security risks while working remotely

1. External attacks

2. Covid 19 Phishing Scams

3. Fake phone calls or messages

4. Bad security practices

5. Infrastructure issues

Remote working best practices

1. Securing your home network

2. Use a VPN

3. Keep your software and systems updated

4. Using MFA (Multi-Factor Authentication)

5. Backup your data
